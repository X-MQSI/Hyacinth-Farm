# Hyacinth Farm Monitor - 交互式 API 工具

## 功能特点

这是一个功能完整的交互式命令行工具，用于测试和调试 Hyacinth Farm Monitor 的所有 API 接口。

### 主要功能

1. **心跳接口** ❤️
   - 发送单次心跳
   - 查询心跳状态
   - 启动心跳循环（每5秒自动发送）

2. **传感器数据接口** 📊
   - 发送预设数据（快速测试）
   - 自定义发送数据（手动输入各项参数）
   - 查询最新数据
   - 按时间范围查询

3. **事件接口** 🔔
   - 发送预设事件（浇水、故障、启动等）
   - 自定义发送事件
   - 查询事件列表

4. **调试日志接口** 🐛
   - 发送各级别日志（DEBUG/INFO/WARN/ERROR）
   - 自定义日志内容
   - 查询和过滤日志

5. **图片接口** 📷
   - 上传图片（指定本地文件路径）
   - 查询图片列表
   - 分页查询

6. **服务状态查询** ℹ️
   - 查看服务器运行状态

7. **完整 API 测试** 🧪
   - 自动运行所有接口测试
   - 显示测试报告

## 使用方法

### 启动工具

```powershell
# 默认连接本地服务器
.\api_tool.ps1

# 连接指定服务器
.\api_tool.ps1 -BaseUrl "http://192.168.1.100:3000"
```

### 操作说明

1. 启动后会显示主菜单
2. 输入数字选择功能
3. 在子菜单中继续选择具体操作
4. 输入 0 返回上级菜单或退出

### 使用示例

#### 示例 1：发送心跳

```
主菜单 → 1 (心跳接口) → 1 (发送心跳)
```

#### 示例 2：自定义传感器数据

```
主菜单 → 2 (传感器数据) → 2 (自定义发送)
然后输入各项参数（留空使用默认值）
```

#### 示例 3：上传图片

```
主菜单 → 5 (图片接口) → 1 (上传图片)
输入图片路径，例如: C:\Users\xxx\Desktop\test.jpg
```

#### 示例 4：启动心跳循环

```
主菜单 → 1 (心跳接口) → 3 (启动心跳循环)
按 Ctrl+C 停止循环
```

## 快速测试流程

1. 启动服务器：`node server.js`
2. 启动工具：`.\api_tool.ps1`
3. 选择功能 7 运行完整测试
4. 查看测试报告确认所有接口正常

## 常见问题

**Q: 提示连接失败？**
A: 检查服务器是否启动，端口是否正确

**Q: 图片上传失败？**
A: 确认文件路径正确，文件格式为 jpg/jpeg/png

**Q: 如何停止心跳循环？**
A: 按 Ctrl+C 键

**Q: 如何修改服务器地址？**
A: 使用参数启动：`.\api_tool.ps1 -BaseUrl "http://your-server:3000"`
